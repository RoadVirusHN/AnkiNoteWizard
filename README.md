# Anki Card Wizard 🪄

> 웹 페이지 정보를 스캔해 간단하고 빠르게 [Anki](https://apps.ankiweb.net/) Note를 추가,생성하는 크롬 확장 프로그램.

## 💡Motivation

어느 날, 영단어의 발음기호와 예시 문장, 그림이 포함된 제대로 된 플래쉬 카드 하나를 만드는데, 숙달되어 평균 3분 정도의 시간이 걸리게 되었고, 하루에 영단어 30개만 외운다고 해도, 무려 매일 90분 정도의 시간을 소모하게 된다는 것을 깨달았습니다.

따라서 해당 과정을 좀 더 효율적으로 자동화하기 위해 Anki Card Wizard를 개발하게 되었습니다. 

현재는 개인적인 사용이 주 목적이지만, 후일 많은 사람들을 위해 배포하고 싶습니다.

## 🛠️ Features

### 카드 스캔 및 추가/수정 기능

_웹 페이지 내에서 정보를 Anki 앱에 Note로 쉽고 빠르게 추가해보세요!_

**카드**는 미리 설정한 템플릿에 따라 웹 페이지 내의 정보를 탐색 및 Note로 가공한 결과입니다.
- 카드는 추가하기 전에 미리보기 후, 원한다면 내용을 수정하실 수 있습니다.

### 템플릿 추가/수정 기능

_Anki Note를 웹 페이지 내 스캔하고, 자동 생성하기 위한 정보를 입력하세요!_

**템플릿**은 Anki Note를 자동 생성 및 웹 페이지 내 탐색하기 위한 정보 입니다. 

이러한 템플릿을 정의하여 템플릿과 일치하는 HTML 정보를 [AnkiConnect](https://ankiweb.net/shared/info/2055492159) Add on을 통해 Anki note로 추가할 수 있습니다.

#### 템플릿의 요소

##### 메타 데이터
템플릿명, 템플릿 ID, 만든이, 버전, 설명, 공유 url 등

##### 탐색 데이터
웹 페이지 내에 카드 탐색을 위해 필요한 데이터들
- 모델명 : 추가될 카드들의 Anki 모델명
  - 현재는 Basic, Cloze, Basic(and reversed card) 모델을 지원합니다.
- URL 패턴 : 현재 웹 페이지 URL과 일치해야만 해당 카드를 탐색합니다.
  - 와일드카드를 이용하여 작성합니다.
  - 기본값: * (모든 웹페이지에서 탐색)
- 루트 태그 : 카드 탐색을 위한 최상단 부모가 되는 HTML 태그의 셀렉터, 기본값: body 태그
  - 페이지 내에 해당하는 태그가 유일하지 않으면 여러 카드가 탐색됩니다.
  - 현재는 CSS Selector만 지원합니다.


##### 노트 데이터
Anki Note 추가시, 추가될 내용이 될 데이터들
- 태그 : Anki Note 추가 시, 기입된 태그들이 추가됩니다.
- 오디오 : Anki Note Audio로 추가될 내용. (현재는 지원하지 않습니다.)
- 필드 : 각 Anki Note 추가 시, 카드의 구성요소가 될 나뉘어진 HTML 코드
